Settings

Cloud: https://router.huggingface.co/v1. Local Ollama: http://localhost:11434/v1 (no token needed; run ollama serve and pull your models first).

Stored locally on this machine only. For Hugging Face, create one (with the “Make calls to Inference Providers” permission) at huggingface.co/settings/tokens.

The vision model describes the photos; then the reasoning model deduces the location from that description. The vision one must be image-capable; the reasoning one can be text-only.
Ollama examples: vision qwen2.5vl · reasoning qwen2.5 (pull with ollama pull qwen2.5vl).

When on, the app looks up the signs/names the model reads (street & business names, landmarks) using OpenStreetMap, Wikipedia & DuckDuckGo to help pin the spot. Those text clues are sent to those services; your photos are not.

Highlight clues for the AI

Drag on the image to box areas the AI should focus on — signs, license plates, shopfronts, landmarks. Each box is sent as a zoomed-in close-up.
